## About

This repository serves as a learning hub for developers interested in generative AI, including:
- LLM (Large Language Model) fundamentals
- Prompt engineering
- RAG (Retrieval-Augmented Generation)
- Fine-tuning and model adaptation
- AI-powered development workflows

## Getting Started

### Prerequisites

- Python 3.8 or higher
- Git
- API keys for relevant AI services (OpenAI, Anthropic, etc.)

### Installation

```bash
# Clone the repository
git clone https://github.com/nautirishu/GenAI-Developer-Learning.git

# Navigate to the project directory
cd GenAI-Developer-Learning

# Install dependencies
pip install -r requirements.txt
```
